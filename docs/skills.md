# Agent Skills

Project skills live under `.claude/skills/` for Claude Code style agents and `.agents/skills/` for Cursor-style agents.

## Issue Plan Observer

`.claude/skills/issue-plan-observer/SKILL.md` and `.agents/skills/issue-plan-observer/SKILL.md` define a neutral review workflow for GitHub issues and implementation plans. Use it when asking an agent to act as an impartial observer, inspect an issue and its plan, compare the plan against repository evidence, and surface risks, mismatches, hidden dependencies, test gaps, and practical suggestions.
